## Step:1 Generate a strong secret
```text
Linux: openssl rand -hex 32
Python:
  import secrets
  print(secrets.token_hex(32))
```
## Step:2 🌍 Step 2: Add Secret in Amazon CloudFront  
Go to:
  CloudFront Console
  Your Distribution
  Origins → Edit

Add Custom Header  
| Field       | Value                     |
| ----------- | ------------------------- |
| Header Name | `X-Origin-Secret`         |
| Value       | `<your-generated-secret>` |

👉 Example:  
```text
X-Origin-Secret: 9f8c7a2b4d5e6fabc123...
```
✔ Save changes  
✔ Wait for distribution to deploy (~5–10 min)  

## ⚙️ Step 3: Store Secret in Kubernetes (BEST PRACTICE)  
Don’t hardcode in code ❌  

Create Kubernetes Secret  
```bash
kubectl create secret generic api-gateway-secret \
  --from-literal=origin_secret=9f8c7a2b4d5e6fabc123...
```
Use it in Deployment  
```yaml
env:
  - name: ORIGIN_SECRET
    valueFrom:
      secretKeyRef:
        name: api-gateway-secret
        key: origin_secret
```

## 🧠 Step 4: Use Secret in FastAPI  
Update your code:  
```python
import os

SECRET_HEADER = os.getenv("ORIGIN_SECRET")

@app.middleware("http")
async def verify_origin(request: Request, call_next):
    if request.headers.get("X-Origin-Secret") != SECRET_HEADER:
        raise HTTPException(status_code=403, detail="Forbidden")

    return await call_next(request)
```
```python
from fastapi import FastAPI, Request, HTTPException

app = FastAPI()

SECRET_HEADER = "my-super-secret-123"

@app.middleware("http")
async def verify_origin(request: Request, call_next):
    origin_secret = request.headers.get("X-Origin-Secret")

    if origin_secret != SECRET_HEADER:
        raise HTTPException(status_code=403, detail="Forbidden: Direct access not allowed")

    response = await call_next(request)
    return response
```
